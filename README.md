# Zamin X

Zamin X نسل جدیدی از فریم‌ورک زمین است که با تمرکز بر سادگی، ماژولار بودن و توسعه‌پذیری بازطراحی شده است.

این پروژه با هدف ارائه مجموعه‌ای از Building Blockها، الگوهای کاربردی و foundationهای آماده برای ساخت سیستم‌های مدرن بر پایه .NET طراحی شده است.

---

## هدف پروژه

Zamin X تلاش می‌کند:

* پیچیدگی‌های تکراری در توسعه نرم‌افزار را کاهش دهد
* یک معماری یکدست و قابل توسعه ارائه دهد
* امکان انتخاب و جایگزینی پیاده‌سازی‌ها را فراهم کند
* توسعه سیستم‌های ماژولار و قابل نگهداری را ساده‌تر کند

---

## فلسفه طراحی

### 1. جداسازی قرارداد از پیاده‌سازی

تمام قراردادها (Abstractions) از پیاده‌سازی‌ها جدا هستند تا:

* وابستگی‌ها کاهش یابد
* تست‌پذیری افزایش یابد
* امکان جایگزینی implementation فراهم شود

---

### 2. ماژولار بودن

هر capability به صورت مستقل طراحی می‌شود:

* هر بخش مانند یک mini-repository است
* دارای solution، README و ساختار مستقل است
* قابلیت توسعه و نگهداری جداگانه دارد

---

### 3. مالکیت در سطح پروژه

هر پروژه مسئول موارد زیر است:

* پیاده‌سازی
* registration (Dependency Injection)
* تنظیمات مرتبط با خودش

---

### 4. سادگی در استفاده

APIها به گونه‌ای طراحی شده‌اند که:

* استفاده اولیه ساده باشد
* در صورت نیاز قابل گسترش باشند

---

## ساختار پروژه

ساختار کلی پروژه به صورت لایه‌ای تعریف شده است:

```text
src/
  00.BuildingBlocks
  01.ApplicationPatterns
  02.Integrations
  03.Foundations
```

### توضیح لایه‌ها

* **BuildingBlocks**
  اجزای پایه و قابل استفاده مستقل (مثل Mapper، Translator و ...)

* **ApplicationPatterns**
  الگوهای رایج در سطح اپلیکیشن (مثل CQRS، Event Handling و ...)

* **Integrations**
  یکپارچه‌سازی با ابزارها و لایبرری‌های خارجی

* **Foundations**
  ساختارهای آماده برای ساخت اپلیکیشن (مثل Web API، Modular Monolith و ...)

---

## نمونه پیاده‌سازی فعلی

در حال حاضر اولین capability پیاده‌سازی شده:

### Mapper

* تعریف قرارداد مستقل برای mapping
* پیاده‌سازی بر پایه AutoMapper
* امکان جایگزینی provider
* دارای sample عملیاتی با Web API

---

## نحوه استفاده (نمونه)

```csharp
builder.Services.AddZaminXAutoMapperAdapter();
```

یا با تنظیمات:

```csharp
builder.Services.AddZaminXAutoMapperAdapter(options =>
{
    options.Assemblies.Add(typeof(Program).Assembly);
});
```
