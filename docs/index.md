# Docs Index

## هدف این پوشه
این پوشه منبع اصلی مستندات Zamin X است.

در اینجا مسیر فهم محصول، معماری، ماژول‌ها، تصمیم‌ها و backlog پروژه نگهداری می‌شود.

این فایل نقطه ورود اصلی برای مطالعه مستندات است و به شما کمک می‌کند:
- ساختار مستندات را بفهمید
- مسیر مطالعه مناسب را پیدا کنید
- بدانید هر سؤال را باید در کدام بخش دنبال کنید

## وضعیت فعلی مستندات
Zamin X در فاز طراحی محصول و معماری قرار دارد.

در این مرحله تمرکز روی موارد زیر است:
- تعریف محصول (Product Definition)
- تثبیت معماری (Architecture Design)
- تعریف taxonomy ریپو
- مشخص کردن مرز ماژول‌ها
- ثبت تصمیم‌های کلیدی (ADR)
- ساخت backlog اولیه

برای مشاهده آخرین وضعیت پروژه و تصمیم‌ها:
👉 `02.architecture/project-state.md`
