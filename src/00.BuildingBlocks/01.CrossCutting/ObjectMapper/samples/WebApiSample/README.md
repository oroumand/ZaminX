# Object Mapper Web API Sample

این پروژه یک نمونه اجرایی برای نمایش استفاده از Object Mapper است.

## هدف این sample

هدف این sample:
- نمایش registration provider
- نمایش inject شدن قرارداد capability
- نمایش scan شدن assemblyهای مورد نیاز
- نمایش یک سناریوی ساده mapping

## نقش این پروژه

این پروژه برای درک سریع usage نگه داشته می‌شود و مرجع اصلی تصمیم‌های معماری capability نیست.
