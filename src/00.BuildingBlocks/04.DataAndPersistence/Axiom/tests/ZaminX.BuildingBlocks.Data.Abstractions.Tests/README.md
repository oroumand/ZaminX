# ZaminX.BuildingBlocks.Data.Abstractions.Tests

این پروژه تست‌های مربوط به abstractionهای مشترک Axiom را نگه می‌دارد.

## مسئولیت‌ها

این پروژه محل تست این موارد است:

* مدل‌های paging
* رفتار normalize شدن `PagedQuery`
* قراردادهای مشترک data abstractions
* رفتار پیش‌فرض `DefaultDataAuditContext`

## هدف

هدف این پروژه اطمینان از پایداری contractهای پایه Axiom است؛ چون هر تغییری در این لایه روی read و write side اثر مستقیم دارد.
