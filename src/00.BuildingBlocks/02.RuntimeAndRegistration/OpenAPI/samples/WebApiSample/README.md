# WebApiSample

این sample یک Web API ساده برای نمایش نحوه استفاده از `Lumen` و UIهای مرتبط است.

---

## آنچه این sample نشان می‌دهد

* registration هسته OpenAPI
* اضافه کردن هم‌زمان Scalar، Swagger و Redoc
* استفاده از configuration برای options
* اجرای runtime setup با یک entry point واحد

---

## run

پس از restore و build، برنامه را اجرا کنید و مسیرهای زیر را بررسی کنید:

* `/openapi/v1.json`
* `/scalar`
* `/swagger`
* `/redoc`

---

## نکته

این sample برای نمایش wiring و setup نوشته شده و intentionally ساده نگه داشته شده است.
